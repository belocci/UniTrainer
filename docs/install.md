# Installation (Windows)

Uni Trainer is a desktop application. End users do not need a CLI.

## Install via Installer
1. Download the Windows installer from the GitHub Releases page.
2. Run the installer and follow the prompts.
3. Launch Uni Trainer from the Start Menu or desktop shortcut.

## Notes
- If Windows SmartScreen prompts, click "More info" then "Run anyway".
- No command line setup is required for end users.
